Emoji-App-Static
----------------

:comet: Static pages for Emoji Generator apps

## Requirements

- Node `$(cat .node-version)`
- Yarn

## Getting Started

```
$ yarn
$ yarn run build
```

## Development

```
# watch & compile PostCSS files
$ yarn run watch

# run development HTTP server
$ yarn start
```
